# Native-and-AI-Assisted-Outlier-Detection-in-a-Glass-Project-Dataset


The data used in this study were generated using the scripts leitura_dados_oxidos and estudo_refractiveindex(nd)_oxidos, both available in the “getting_data” folder of this repository. By first running leitura_dados_oxidos and then estudo_refractiveindex(nd)_oxidos, the clean refractive index data can be obtained. These processed data are stored in the RefractiveIndex_clean folder within “data” and are used by all scripts in the “code” folder, which contain the experimental results of this research.
